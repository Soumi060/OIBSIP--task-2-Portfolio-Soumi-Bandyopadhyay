💼 Personal Portfolio Website (Level 1. Task 2)

📌 Overview

I built this personal portfolio website to showcase my skills and experience. It has my info, skills, projects and contact details in a clean format.

The main goal is to have a presence and show my frontend development skills using HTML and CSS.

🚀 Features

* A section that introduces me

* A showcase of my skills

* A section that displays my projects

* Integration with my resume (optional)

* A section with my contact information

* A clean design

🛠️ Technologies Used

* HTML5 – for the page structure

* CSS3 – for styling and layout

📚 What I Learned

* How to design a personal portfolio layout

* How to structure content effectively

* How to style with CSS for presentation

* Creating sections like About, Skills, Projects and Contact

* Improving basic UI/UX
